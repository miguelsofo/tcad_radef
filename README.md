# Codigos TCAD de ejemplo para la materia de: Efectos de la radiacion en CI. 

Antes de empezar, hacer:
```
source source_sentaurus 
```

Carpetas con archivos de comandos: 
* cmds
* svisual
* structure
* plots

Carpetas con archivos de salida:
* logs
* plts
* currents
* savs
* out_svisual 
* results

Para ejecutar un script de svisual hacer:
```
svisual -bx script.tcl 
```

En la carpeta plots estan los scripts de python para graficar datos.
